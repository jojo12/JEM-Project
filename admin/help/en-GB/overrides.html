Module Template Overrides

1. Identify the module layout you want to override

Each JEM module has its layout files in: /modules/mod_jem_xxx/tmpl/ For example: default.php, table.php, etc.

2. Create the override via Joomla backend (recommended)

  1. Open Joomla administrator.
  2. Go to System → Site Templates.
  3. Select your active site template.
  4. Open the Create Overrides tab.
  5. Under Modules, select the JEM module (e.g. mod_jem_teaser). Joomla will automatically copy the layout files into your template folder: /templates/YOUR_TEMPLATE/html/mod_jem_teaser/

3. Create the override manually (alternative)

Copy the layout file from: /modules/mod_jem_teaser/tmpl/default.php to: /templates/YOUR_TEMPLATE/html/mod_jem_teaser/default.php

4. Edit the override

Open the copied file inside your template folder and modify it as needed. Joomla will use this file instead of the module’s original layout.

5. Optional: CSS override

If the module loads its own CSS, you can override it by placing a CSS file in your template, for example: /templates/YOUR_TEMPLATE/css/mod_jem_teaser.css.

6. After updates

When JEM or the module is updated, compare new module layout files with your overrides and adjust if required.
